# SOQL Explorer

Build, run, and save SOQL queries with schema-aware assistance and exports.

## Where to find
- Developer → SOQL Explorer

## Key features
- Describe-driven SObject and Fields panels with search and relationships
- Monaco editor with tab support
- Execute queries (supports Tooling API toggle and Include Deleted Records)
- Explain plan (Performance check)
- Export results (CSV/TSV/JSON) and Copy to clipboard
- Select rows to perform actions (e.g., delete)
- Recent queries and Saved queries (Global / Org-specific)

## Persistence
- Recent queries: local per org
- Saved queries:
  - Global: available to all orgs
  - Org-specific: scoped to current org

## Tips
- Use the SObjects and Fields panels to insert fields quickly
- Use Performance check before exporting large datasets
- Saved queries appear grouped by scope in the left “Saved” list
