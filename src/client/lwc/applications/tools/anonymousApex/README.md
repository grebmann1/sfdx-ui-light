## Key features
- Monaco-based Apex editor with tabs
- Execute Anonymous Apex
- Debug log viewer with filters (User Debug, etc.)
- Inline error markers for compile/runtime issues
- Recents and Saved scripts (Global / Org-specific)

## Persistence
- Recent scripts: local per org
- Saved scripts:
  - Global: available to all orgs
  - Org-specific: scoped to current org

## Tips
- Use the debug filters to focus on USER_DEBUG
- Save scripts you reuse and organize by Global vs Org
